## 🎯 Project Overview

**Nama Project:** Aplikasi Kalkulator Turunan Fungsi Berbasis Web

**Deskripsi:** Aplikasi interaktif berbasis JavaScript yang menghitung turunan fungsi matematika dan menampilkan langkah-langkah penyelesaian secara detail.

**Tipe Project:** Studi Kasus Aplikasi Pembelajaran (Educational Application)

**Durasi Pengembangan:** Single File Solution (All-in-one HTML)

**Target Audience:** Siswa, Guru, Profesional yang belajar Kalkulus

## 🔧 Fitur Teknis

### Parsing & Computation
- **Parser Type:** Recursive descent parser
- **Data Structure:** Abstract Syntax Tree (AST)
- **Computation:** Symbolic (exact) differentiation
- **Simplification:** Basic algebraic simplification

### Supported Operators
| Operator | Symbol | Contoh |
|----------|--------|--------|
| Power | `^` | `x^2` |
| Multiplication | `*` | `3*x` |
| Division | `/` | `x/2` |
| Addition | `+` | `x+1` |
| Subtraction | `-` | `x-1` |
| Grouping | `()` | `(x+1)^2` |

### Supported Functions
```javascript
sin(x), cos(x), tan(x)    // Trigonometri
exp(x)                     // e^x
log(x)                     // ln(x)
sqrt(x)                    // √x
```

## 🎯 Use Cases

### Untuk Siswa SMA/Universitas
```
Scenario:
"Saya tidak mengerti bagaimana cara menghitung d/dx[sin(x²)]"

Solution:
1. Buka aplikasi
2. Input: sin(x^2)
3. Lihat step-by-step:
   - Chain rule diterapkan
   - Outer function: sin
```

## 💡 Innovation Points

1. **Symbolic Computation**: Exact mathematical computation, bukan numerical approximation

2. **AST-Based Parsing**: Robust parsing dengan Abstract Syntax Tree, bukan regex-based

3. **Educational Focus**: Setiap step dijelaskan dengan rule yang digunakan

4. **No Dependencies**: Pure vanilla JavaScript, tidak perlu install library apapun

5. **Offline Ready**: 100% client-side, bisa digunakan tanpa internet

6. **Responsive Design**: Bekerja di desktop, tablet, dan mobile

7. **Extensible Architecture**: Mudah menambah fungsi/rule baru

## ⚠️ Keterbatasan (Known Limitations)

1. **Input complexity limit**: ~500 karakter untuk optimal performance
2. **u^v (variable exponent)**: Belum fully supported
3. **Multiple variables**: Hanya support turunan terhadap x
4. **Advanced functions**: arcsin, arccos, sinh, cosh, dll belum ada
5. **Simplification**: Basic simplification saja, bukan full algebraic simplification
6. **Numerical evaluation**: Belum ada fitur evaluate f'(a) untuk nilai x tertentu

*Catatan: Semua ini bisa ditambahkan dengan development lanjutan*
